# Market Simulator

A Python-based market simulator that includes:

* Order matching engine
* Trade execution
* Portfolio tracking
* P&L calculation
* Average buy price
* Trade statistics

## Files

* Main.py - main simulator
* market.py - supporting code
* trial.py - testing code

## Example Features

* Buy and sell orders
* Matching highest bid with lowest ask
* Cash management
* Position tracking
* Portfolio valuation
